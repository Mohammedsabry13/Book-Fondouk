# Book Fondouk — Travel Agency Landing Page

⚠️ This template was originally created by Mohammed Sabry credit is required: github.com/Mohammedsabry13


A clean and elegant one-page corporate website for **Book Fondouk**, a UAE-based travel agency (in partnership with Intratour) offering complete travel planning services for individuals and businesses.

## 🛠️ Tech Stack

- **HTML5 & CSS3** — semantic one-page structure and custom styling
- **Bootstrap 5.3** — responsive grid and components
- **Bootstrap Icons** — icon set
- **Mulish** — Google Fonts elegant typography
- **Vanilla JavaScript** — mobile menu, services filter tabs, dynamic cards, year auto-update

## ✨ Features

- **Promo banner** — top "It's time to travel, book now" announcement strip
- **Transparent sticky navbar** — with phone number (+971586712271), smooth scroll links and mobile overlay menu
- **Full-screen hero section** — airport background image with centered circle card, title, subtitle and Book Now CTA
- **Online Appointments section** — JS-powered filterable services grid with 6 category tabs (All, Hotel Booking, Flight Tickets, Honeymoon, MICE, Transportation); cards injected dynamically via JavaScript
- **Scope of Services section** — 3 image+text cards (MICE, VIP Meet & Greet, Transportation)
- **"What We Also Do" section** — 3 fullscreen background overlay cards (Honeymoon Packages, Hotel Bookings, Flight Tickets)
- **Business Partners section** — split layout with benefits text (Saving Time, Reliable Travel, Saving Money) and travel image
- **Connect With Us** — social icons (Facebook, Instagram, LinkedIn)
- **Mailing List signup** — fullscreen background section with email subscription form
- **Footer** — copyright, Terms, Privacy Policy, Payment Confirmation, Refund Policy links + Intratour partnership note
- Fully responsive across all screen sizes
